Configurazione VAPID

Setup una-tantum (chiavi generate localmente, senza account Google):

  1. Genera le chiavi:
    dotnet run -- generate-vapid-keys
  2. Imposta le chiavi via user-secrets con i comandi stampati nel terminale.
  3. Riavvia l'app.
